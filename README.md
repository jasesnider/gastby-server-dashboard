STEPS TO RUN IN DEVELOP

1. Get Gatsby CLI - 

```sh 
npm install --global gatsby-cli
```

2. Clone Repo

```sh
git clone git@github.com:jasesnider/gastby-server-dashboard.git
```

3. CD into app 

```sh
cd gatsby-server-dashboard
```

4. Install dependencies 

```sh
npm install
```

5. Run app in develop

```sh
gatsby develop
```

6. Go to: 

```sh
http://localhost:8000
```
